1 Цель работы: 
1.1 Изучить теорию о методах детектирования.
1.2 Научиться определять значения сигналов по сигнальному созвездию.
1.3 Освоить навыки модуляции детектора на языке Python или MATLAB.

2 Задание:
2.1 Изучить теоретический материал
2.2 Закрепить материал на практике:
2.2.1 Для выбранного типа модуляции рассчитать такое значение SNR, при котором вероятность ошибки будет равна 0.5%. Сравнить расчётное значение SNR с практическим.
2.2.2 По сигнальному созвездию детектировать сигналы с разными типами модуляции и ввести полученные результаты детектирования в поля для ввода.
2.2.3 Смоделировать QPSK детектор на языке Python, в MATLAB или на другом языке программирования. Проверить работу детектора. 
2.2.4 Смоделировать OFDM детектор на языке Python, в MATLAB или на другом языке программирования. Проверить работу детектора.
2.3 Прохойти контрольную проверку освоения материала

3 Порядок выполнения работы:
3.1 Перейдите в раздел "Теория" и изучите теоретический материал, представленный в 3 главах:
	1) Общие теоретические сведения о детектировании 
	2) Классификация методов детектирования 
	3) Современные методы детектирования

3.2 Перейдите в раздел "Практические задания" и выполните задание 1, придерживаясь методических указаний.
3.2.1 Прочитайте задание для данной работы и рассмотрите пример выполнения работы.  
3.2.2 Выберите тип манипуляции (из выпадающего списка), для которого будет сформирован сигнал.
3.2.3 Нажмите на кнопку "Перейти к практике".
3.2.4 Изучите информацию, представленную на странице (Сгенерированный сигнал, который прошёл основные этапы передачи и приёма в системе цифровой связи) 
3.2.5 Рассчитайте и подберите такое значение SNR, при котором вероятность детектирования сообщения без ошибки будет около 0.5%. Введите полученное значение в поле для ввода.
3.2.6 Зафиксируйте полученные вероятности ошибок и значения SNR для данных вероятностей.
3.2.7 Повторите пункты 3.2.2 - 3.2.6 ещё несколько раз, но уже для других видов манипуляции.
3.2.8 Сделайте вывод о влиянии SNR на различные типы модуляции и почему теоретические значения вероятности ошибки отличаются от практических. 

3.3 Перейдите в раздел "Практические задания" и выполните задание 2, придерживаясь методических указаний.
3.3.1 Прочитайте задание для данной работы и рассмотрите пример выполнения работы.
3.3.2 Нажмите на кнопку "Перейти к практике".
3.3.3 Детектируйте 3 представленных сигнала, используя сигнальное созвездие. Результат детектирования записывайте в поле для ввода под каждым созвездием.
3.3.4 Нажмите кнопку "Проверить", чтобы система проверила, правильно ли выполнено детектирование сигналов. 
3.3.5 При успешном выполнении работы, зафиксируйте результат. Поясните процесс детектирования.

3.4 Перейдите в раздел "Практические задания" и выполните задание 3, придерживаясь методических указаний.
3.4.1 Изучите теорию о детектировании фазо-манипулированных сигналов.
3.4.2 Реализуйте на языке Python или MATLAB QPSK детектор. Пример входных и выходных данных указан на странице с заданием.
3.4.3 Для проверки работоспособности детектора, нажмите на кнопку "Проверить". На открывшейся странице будет сгенерировано несколько сигналов, представленных в комплексной форме. Внесите значения сигналов исходные данные для детектирования. Выходные данные внесите в поля для ввода.
3.4.4 При успешном выполнении работы, зафиксируйте результат. Также сохраните код детектора.

3.5 Перейдите в раздел "Практические задания" и выполните задание 4, придерживаясь методических указаний.
3.5.1 Изучите теорию о детектировании OFDM-манипулированных сигналов.
3.5.2 Реализуйте на языке Python, в MATLAB или на другом языке программирования OFDM детектор. Пример входных и выходных данных указан на странице с заданием.
3.5.3 Для проверки работоспособности детектора, нажмите на кнопку "Проверить". На открывшейся странице будет сгенерировано несколько сигналов, представленных в комплексной форме. Внесите значения сигналов исходные данные для детектирования. Выходные данные внесите в поля для ввода.
3.5.4 При успешном выполнении работы, зафиксируйте результат. Также сохраните код детектора.

3.6 Перейдите в раздел "Тестирование".
3.6.1 Для каждого вопроса выберите один вариант ответа.
3.6.2 Нажмите на кнопку "Проверить", для оценки ваших знаний.
3.6.3 При правильном ответе на 80% вопросов, откроется страница с результатом. Зафиксируйте результат.
